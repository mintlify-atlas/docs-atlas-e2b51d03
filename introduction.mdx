## What is VirtualizorJS?

VirtualizorJS is a **TypeScript-first SDK** for the [Virtualizor](https://www.virtualizor.com/) server management API. It provides a clean, type-safe interface for managing VPS instances, users, and plans with zero production dependencies.

Whether you're building a hosting control panel, automating VPS deployments, or integrating Virtualizor into your infrastructure, VirtualizorJS makes it easy with a modern, developer-friendly API.

## Quick Example

```typescript
import { createVirtualizorClient } from 'virtualizorjs';

const client = createVirtualizorClient({
  host: 'virtualizor.example.com',
  apiKey: 'your-api-key',
  apiPass: 'your-api-pass',
});

// List all VPS instances
const vpsList = await client.vps.list();

// Start a VPS and wait for it to complete
const result = await client.vps.start('123');
const task = await client.tasks.wait(result.taskid!);

console.log(`VPS started! Task status: ${task.status}`);
```
