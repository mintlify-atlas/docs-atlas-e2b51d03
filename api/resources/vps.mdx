title: VpsResource
description: Manage virtual private servers (VPS) in Virtualizor

The `VpsResource` class provides methods to create, manage, and monitor VPS instances.

## Methods

### list()

List all VPS instances with optional filters.

<ParamField path="filters" type="ListVPSParams" optional>
  Filtering options for VPS listing
  
  <Expandable title="properties">
    <ParamField path="user" type="string | number">
      Filter by user ID
    </ParamField>
    <ParamField path="vpsid" type="string | number">
      Filter by VPS ID
    </ParamField>
    <ParamField path="vpsname" type="string">
      Filter by VPS name
    </ParamField>
    <ParamField path="vpsip" type="string">
      Filter by IP address
    </ParamField>
    <ParamField path="vpshostname" type="string">
      Filter by hostname
    </ParamField>
    <ParamField path="vsstatus" type="string">
      Filter by status
    </ParamField>
    <ParamField path="vstype" type="VirtType">
      Filter by virtualization type (kvm, openvz, xen, etc.)
    </ParamField>
    <ParamField path="serid" type="number">
      Filter by server ID
    </ParamField>
    <ParamField path="plid" type="number">
      Filter by plan ID
    </ParamField>
    <ParamField path="bpid" type="number">
      Filter by billing plan ID
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="return" type="Promise<Record<string, VPS>>">
  Object mapping VPS IDs to VPS objects
</ResponseField>

```typescript
const allVps = await virtualizor.vps.list();

// Filter by user
const userVps = await virtualizor.vps.list({ user: '123' });

// Filter by virtualization type
const kvmVps = await virtualizor.vps.list({ vstype: 'kvm' });
```

### create()

Create a new VPS instance. Returns an async task result.

<ParamField path="params" type="CreateVPSParams" required>
  VPS creation parameters
  
  <Expandable title="properties">
    <ParamField path="hostname" type="string" required>
      Hostname for the VPS
    </ParamField>
    <ParamField path="rootpass" type="string" required>
      Root password for the VPS
    </ParamField>
    <ParamField path="osid" type="number" required>
      OS template ID to use
    </ParamField>
    <ParamField path="plid" type="number">
      Plan ID to use
    </ParamField>
    <ParamField path="user_email" type="string">
      User email address
    </ParamField>
    <ParamField path="ips" type="number">
      Number of IPv4 addresses
    </ParamField>
    <ParamField path="ips_int" type="number">
      Number of internal IPs
    </ParamField>
    <ParamField path="space" type="number">
      Disk space in GB
    </ParamField>
    <ParamField path="ram" type="number">
      RAM in MB
    </ParamField>
    <ParamField path="burst" type="number">
      Burst RAM in MB
    </ParamField>
    <ParamField path="bandwidth" type="number">
      Bandwidth in GB
    </ParamField>
    <ParamField path="cpu" type="number">
      Number of CPU cores
    </ParamField>
    <ParamField path="cpu_percent" type="number">
      CPU percentage limit
    </ParamField>
    <ParamField path="virt" type="VirtType">
      Virtualization type (kvm, openvz, xen, etc.)
    </ParamField>
    <ParamField path="serid" type="number">
      Server ID to create VPS on
    </ParamField>
    <ParamField path="node_select" type="0 | 1">
      Enable automatic node selection
    </ParamField>
    <ParamField path="recipe" type="number">
      Recipe ID to execute after creation
    </ParamField>
    <ParamField path="sshkey" type="string">
      SSH public key to inject
    </ParamField>
    <ParamField path="nopassword" type="0 | 1">
      Disable password authentication (SSH key only)
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="return" type="Promise<AsyncTaskResult>">
  Async task result with task ID for tracking
</ResponseField>

```typescript
const result = await virtualizor.vps.create({
  hostname: 'web-server-01.example.com',
  rootpass: 'SecureP@ssw0rd',
  osid: 270, // Ubuntu 22.04
  plid: 5,
  space: 50,
  ram: 2048,
  bandwidth: 1000,
  cpu: 2,
  ips: 1,
});

// Wait for creation to complete
const task = await virtualizor.tasks.wait(result.done.taskid);
console.log('VPS created successfully:', task);
```

## Async Operations

Many VPS operations (create, delete, start, stop, restart, poweroff, suspend, unsuspend, rebuild, clone, migrate) return `AsyncTaskResult` objects. Use the `tasks.wait()` method to poll for completion:

```typescript
// Start a VPS and wait for completion
const result = await virtualizor.vps.start('1234');
const task = await virtualizor.tasks.wait(result.done.taskid);

if (task.status === '1' || task.status === 'done') {
  console.log('VPS started successfully');
}
```
